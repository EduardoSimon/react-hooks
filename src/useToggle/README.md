# useToggle

`useToggle` for toggle state

#### Usage

```js
const [on, toggle] = useToggle();
const [on] = useToggle(false);
```

#### Config

|                                | Description |
| :----------------------------- | ----------- |
| Initial state. Default = false |

#### Return pair

| Description                  |
| :--------------------------- |
| Current state value          |
| Function to update the state |
